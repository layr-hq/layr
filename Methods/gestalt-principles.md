# Gestalt Principles

## What
Gestalt Principles describe how users naturally perceive and group visual elements.

Key principles:
- Proximity - elements close together are seen as related  
- Similarity - elements that look alike are grouped together  
- Continuity - users follow visual paths and alignments  
- Figure/Ground - users distinguish foreground from background  

## Why it matters
Users do not read interfaces line by line.  
They scan and interpret structure visually.

Poor grouping creates confusion and increases cognitive load.  
Good grouping creates instant understanding and reduces effort.

## When to apply
- layout structure
- grouping of content and components
- dashboards and data-heavy screens
- forms and input sections
- any screen with multiple elements

## How to apply

- group related elements closely (proximity)
- separate unrelated elements with space
- use consistent styles for related elements (similarity)
- align elements to create clear visual paths (continuity)
- ensure primary content stands out clearly from background (figure/ground)

Structure should be understood before reading.

## Implementation rules

- use spacing to define relationships (not borders alone)
- maintain consistent styles for similar elements
- align elements to a grid for predictable scanning
- avoid random placement or uneven spacing
- clearly separate sections using whitespace
- ensure primary content is visually dominant

## Example

Bad:
- evenly spaced elements with no grouping
- inconsistent styles across related items
- cluttered layout with no clear structure

Good:
- clear sections with grouped content
- consistent styling for related elements
- strong visual hierarchy and alignment

## Fail conditions

- user cannot tell what belongs together
- layout feels random or unstructured
- user must read everything to understand relationships
- unclear separation between sections

## Enforcement rule

If relationships between elements are not instantly clear, restructure using spacing, alignment, and consistency until grouping is obvious.
