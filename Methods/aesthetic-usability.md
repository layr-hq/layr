# Aesthetic-Usability Effect

## What it is
Users perceive visually pleasing designs as easier to use.

## Why it matters
Good design increases trust and tolerance for minor issues.

## How to apply
- maintain clean, consistent UI
- use spacing and hierarchy properly
- avoid clutter and noise

## Example
Bad:
Messy, inconsistent UI

Good:
Clean, structured, balanced layout

## Fail signs
- UI feels chaotic
- user doesn’t trust the product

## Rule
If it looks better, it feels easier.
