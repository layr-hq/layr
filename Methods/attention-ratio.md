# Attention Ratio

## What
Attention Ratio is the number of elements competing for a user’s attention versus the one action they should focus on.

The higher the number of competing elements, the lower the clarity and likelihood of action.

## Why it matters
Users have limited attention.  
When too many elements compete, focus is diluted, decision-making slows, and conversion drops.

Clear focus increases speed, confidence, and action.

## When to apply
- landing pages
- onboarding screens
- dashboards
- forms and flows
- any screen with a primary action

## How to apply

- define one clear primary action per screen
- reduce or remove competing actions
- visually prioritise the primary action using contrast, size, and spacing
- minimise secondary actions and lower their emphasis
- remove unnecessary links, buttons, or distractions

Design should guide attention, not divide it.

## Implementation rules

- one primary action per screen
- maximum 1–2 secondary actions
- primary action must have highest visual contrast
- secondary actions must have lower visual weight
- avoid multiple elements with equal emphasis
- limit visible actions to what is necessary for the next step

## Example

Bad:
- multiple buttons with equal size and styling
- navigation, links, and actions competing for attention
- no clear focal point

Good:
- one dominant primary action
- secondary actions visually reduced
- clear visual hierarchy guiding focus

## Fail conditions

- user scans multiple areas before acting
- user hesitates or delays action
- unclear what the next step is
- multiple elements appear equally important

## Enforcement rule

If multiple elements compete for attention, reduce until the next action is obvious and dominant.
