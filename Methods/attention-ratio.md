# Attention Ratio

## What it is
The number of things competing for a user’s attention vs the one thing they should focus on.

## Why it matters
More distractions = lower conversion and more confusion.

## How to apply
- one primary action per screen
- reduce competing elements
- remove unnecessary links and options
- use contrast to guide focus

## Example
Bad:
Multiple buttons with equal importance

Good:
One clear primary action, everything else secondary

## Fail signs
- user doesn’t know where to click
- user scans repeatedly
- low interaction rates

## Rule
The fewer distractions, the clearer the action.
