# Journey Friction

## What it is
Anything that slows down or blocks user progress.

## Why it matters
Friction reduces conversion and increases drop-off.

## How to apply
- remove unnecessary steps
- reduce form fields
- simplify flows
- eliminate delays

## Example
Bad:
Multiple steps with unclear purpose

Good:
Direct path to completion

## Fail signs
- users drop off mid-flow
- users abandon tasks

## Rule
Make the path to value as short as possible.
