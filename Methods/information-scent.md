# Information Scent

## What
Information Scent refers to how users use cues such as labels, buttons, and links to predict what will happen next.

Users follow signals that indicate value and direction.

## Why it matters
If labels or actions are unclear, users cannot predict outcomes.

This leads to:
- hesitation
- wrong actions
- navigation errors

Clear cues guide users confidently.

## When to apply
- navigation menus
- buttons and CTAs
- links and labels
- multi-step flows

## How to apply

- use clear, descriptive labels
- ensure actions match user expectations
- avoid vague or generic wording
- make next steps obvious through wording

Users should understand outcomes before clicking.

## Implementation rules

- use action-based labels (e.g. "Create Account", not "Continue")
- avoid ambiguous terms
- ensure consistency across similar actions
- align wording with user intent
- make next steps explicit

## Example

Bad:
- vague labels like "Submit" or "Next"
- unclear navigation

Good:
- descriptive actions like "Start Free Trial"
- clear navigation paths

## Fail conditions

- user unsure what will happen
- incorrect actions taken
- hesitation before clicking

## Enforcement rule

If a user cannot predict the outcome of an action, rewrite the label until it clearly communicates the result.
