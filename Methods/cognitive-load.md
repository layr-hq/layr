# Cognitive Load Theory

## What it is
Users have limited mental capacity. The more they have to think, the worse the experience.

## Why it matters
High cognitive load makes products feel slow, confusing, and exhausting.

## How to apply
- remove unnecessary decisions
- avoid dense text and clutter
- break tasks into small steps
- use familiar patterns (don’t reinvent)

## Example
Bad:
User must read, interpret, and decide everything

Good:
User is guided step-by-step with minimal thinking

## Fail signs
- user hesitates or re-reads
- user feels overwhelmed
- user abandons flow

## Rule
If the user has to think, simplify.
